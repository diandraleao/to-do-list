# 📝 To-do List 
This is a to-do list in React. The layout is a mix of Dribbble searches.

## 💡 Tecnologies
- Javascript
- React
- Styled-components
- Webpack

## ⚡️ Install dependencies 

```sh
npm install
```

## ▶️ Execute project  

```sh
npm start
```
You can open at
[http://localhost:3000/](http://localhost:3000/)
